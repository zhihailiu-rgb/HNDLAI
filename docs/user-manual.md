## 1. 系统概述

### 1.1 系统简介

本系统是为**海南省建设工程定额人工费调整指数**项目配套开发的管理平台，由**海南省建设标准定额站**主导建设。系统覆盖从数据采集、指数计算到成果发布的完整业务流程，旨在为工程造价管理提供科学、高效的信息化工具。

### 1.2 核心功能

| 模块 | 功能说明 |
|------|----------|
| **仪表盘** | 综合数据看板，展示关键指标和趋势图表 |
| **项目管理** | 管理建设工程项目的基本信息 |
| **数据管理** | 录入和审核各工种劳务市场价格数据 |
| **指数管理** | 一键计算人工费调整指数，管理发布状态 |

### 1.3 技术架构

- **后端**：Python Flask + SQLAlchemy ORM
- **前端**：Bootstrap 5 + ECharts 5
- **数据库**：SQLite（开发环境）/ MySQL（生产环境）

## 2. 快速开始

### 2.1 环境要求

| 软件 | 最低版本 |
|------|----------|
| Python | 3.8 及以上 |
| pip | 20.0 及以上 |
| 浏览器 | Chrome 90+ / Edge 90+ / Firefox 90+ |

### 2.2 安装与启动

```bash
# 1. 获取代码
git clone https://github.com/zhihailiu-rgb/HNDLAI.git
cd HNDLAI

# 2. 安装依赖
pip install flask flask-sqlalchemy pymysql flask-cors

# 3. 生成模拟数据（首次使用）
python seed_data.py

# 4. 启动系统
python run.py
```

启动后，浏览器访问 **http://127.0.0.1:5000** 即可进入系统。

### 2.3 注意事项

- 系统默认使用 **SQLite** 数据库，数据存储在 `data.db` 文件中
- 预置的模拟数据包含 12 个工程项目、9 类工种的 3240 条价格记录和 1080 条指数结果
- 关闭终端窗口即停止服务，重启后数据不丢失

## 4. 项目管理

### 4.1 功能概述

管理所有建设工程项目的基础信息，作为后续数据录入和指数计算的载体。

### 4.2 查看项目列表

进入"项目管理"页面，系统以表格形式展示所有项目，支持分页浏览。表格字段包括：

| 字段 | 说明 |
|------|------|
| 项目名称 | 建设工程的全称 |
| 工程类型 | 住宅/医院/学校/市政 |
| 所在城市 | 项目所在地 |
| 建筑面积 | 项目的总建筑面积（㎡） |
| 总用工量 | 项目预计总用工量（工日） |
| 创建时间 | 数据录入系统的日期 |

### 4.3 新增项目

1. 点击页面右上角 **"新增项目"** 按钮
2. 在弹出的表单中填写项目信息
3. 带 `*` 的字段为必填项
4. 点击 **"保存"** 完成新增

### 4.4 编辑项目

1. 在项目列表中找到目标项目
2. 点击操作列的 **编辑图标**（✏️）
3. 修改需要更新的字段
4. 点击 **"保存"** 提交修改

### 4.5 删除项目

1. 点击操作列的 **删除图标**（🗑️）
2. 系统弹出确认对话框
3. 确认后，该项目及其关联的价格记录和指数结果将被一并删除

> ⚠️ 删除操作不可恢复，请谨慎操作。

## 5. 数据管理

### 5.1 功能概述

数据管理是本系统的**核心模块**，用于采集和审核各工种在不同项目中的劳务市场价格数据，为指数计算提供基础数据支撑。

### 5.2 筛选数据

页面上方提供两个筛选条件：

| 筛选条件 | 说明 |
|----------|------|
| 项目筛选 | 按特定项目查看价格数据 |
| 工种筛选 | 按特定工种查看价格数据 |

选择条件后，系统自动刷新列表。筛选结果同样支持分页浏览。

### 5.3 录入价格数据

1. 点击 **"录入价格"** 按钮
2. 选择对应的 **项目** 和 **工种**
3. 填写 **市场单价**（元/工日，可保留两位小数）
4. 选择 **数据日期**
5. 选择 **数据来源**（劳务分包结算 / 市场询价 / 广联达计价 / 其他）
6. 可填写备注信息
7. 点击 **"保存"** 完成录入

### 5.4 审核数据

为保证数据质量，系统支持审核机制：

- **待审核**：新录入的数据默认为待审核状态（黄色标签）
- **已审核**：点击 **审核图标**（✓）切换状态（绿色标签）
- 建议由专人负责数据审核，确保价格数据的准确性

### 5.5 价格趋势图

页面下方的趋势图展示各工种市场单价随时间的变化情况：

- 可选择查看 **全部工种** 或 **特定工种**
- 折线图直观反映价格波动趋势
- 多工种可叠加显示，便于横向比较

### 5.6 数据采集说明（参考）

根据《海南省建设工程定额人工费调整指数工作方案》，数据采集需注意：

1. **覆盖区域**：海口、三亚、儋州、文昌、琼中、乐东
2. **工程类型**：住宅、医院、学校、市政，每类不少于 2 个项目
3. **工种范围**：覆盖 9 类主要工种
4. **采集频率**：按月采集，按季度汇总计算
5. **季节性因素**：4-10 月为施工旺季，价格通常略高于淡季

## 6. 指数管理

### 6.1 指数计算原理

指数采用"定基指数法"计算，公式如下：

```
指数 = (Pa / Pj) × 100
```

其中：
- **Pa**：报告期平均人工单价（该季度市场采集数据加权平均）
- **Pj**：基期人工单价（琼建规〔2022〕3号，148元/工日）

**解读**：
- 指数 = 100：人工费与基期持平
- 指数 > 100：人工费较基期上涨（例如指数=120，表示上涨了20%）
- 指数 < 100：人工费较基期下跌

### 6.2 综合概览

进入"指数管理"页面，顶部展示两个图表：

1. **各期综合指数概览**：柱状图展示各年季度的人工费综合指数
2. **各工种平均指数**：对比不同工种的指数水平

### 6.3 查询指数

通过年、季度、项目、工种等条件筛选查询指数结果。

### 6.4 计算指数

1. 点击 **"计算指数"** 按钮
2. 选择要计算的 **年份** 和 **季度**
3. 点击 **"开始计算"**
4. 系统自动获取该季度的市场均价，与基期价格对比计算
5. 计算完成后显示结果数量

> 提示：如果某项目某工种在该季度没有价格数据，系统会自动跳过，不会生成无效指数。

### 6.5 发布指数

指数计算完成后，可选择发布：

- **单条发布**：在指数列表中点击操作列的 **发布图标**
- **批量发布**：点击 **"批量发布"**，选择年份和季度，一键发布该期所有指数

发布后的指数会带上发布日期，状态变为"已发布"。已发布的指数不可重复发布。

## 7. 常见问题

### Q1：系统无法启动怎么办？

检查 Python 和依赖包是否安装正确：

```bash
python --version
pip list | findstr flask
```

如缺少依赖，执行：

```bash
pip install flask flask-sqlalchemy pymysql flask-cors
```

### Q2：如何切换到 MySQL 数据库？

```bash
# 设置环境变量
set DB_TYPE=mysql
set MYSQL_HOST=localhost
set MYSQL_PORT=3306
set MYSQL_USER=root
set MYSQL_PASS=your_password
set MYSQL_DB=hainan_labor_index

# 启动系统
python run.py
```

### Q3：如何重置数据？

直接删除 `data.db` 文件，然后重新运行：

```bash
python seed_data.py
```

### Q4：浏览器打开后页面显示异常？

- 推荐使用 Chrome 或 Edge 最新版本
- 确保网络畅通（页面使用了 CDN 加载 Bootstrap 和 ECharts）
- 如无法加载 CDN 资源，可尝试刷新页面

### Q5：如何修改基期价格？

修改 `seed_data.py` 中 `QuotaBase` 部分的 `base_price` 值，然后重新运行：

```bash
python seed_data.py
```

### Q6：系统支持哪些浏览器？

推荐 Chrome 90+、Edge 90+、Firefox 90+。系统基于 Bootstrap 5 开发，兼容主流现代浏览器。
